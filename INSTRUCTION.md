# Инструкция по работе с приложением PC Builder PRO

## Установка и запуск

```bash
pip install -r requirements.txt
python seed.py      # наполнить БД (один раз)
python app.py       # запуск сервера
```
Приложение доступно по адресу: http://127.0.0.1:5000

---

## Сценарий 1. Работа от пользователя

### 1.1. Главная страница (/)
- Отображаются категории комплектующих (CPU, GPU, RAM, Motherboard, PSU, Storage) с количеством товаров в каждой.
- Нажатие на категорию переводит в каталог с фильтром.

### 1.2. Каталог (/catalog)
- Таблица всех комплектующих с характеристиками: название, категория, сокет, TDP, цена.
- Фильтрация по категориям (кнопки сверху).
- Кнопка «⟳ цена» — обновляет цену конкретного товара с интернет-магазина.
- Кнопка «⟳ Обновить все цены» — обновляет цены всех товаров разом.

### 1.3. Конструктор сборки (/build)
1. Выберите комплектующие из выпадающих списков (CPU, Motherboard, GPU, RAM, PSU, Storage).
2. Нажмите «Рассчитать».
3. Результат:
   - **Итоговая цена** сборки.
   - **Потребление** — суммарный TDP всех компонентов (Вт).
   - **Рекомендуемый БП** — мощность блока питания с запасом 30%.
   - **Проверка совместимости** — предупреждение, если сокет CPU не совпадает с сокетом материнской платы.
   - **Ожидаемый FPS** — прогноз кадров в секунду в популярных играх (Cyberpunk 2077, CS2, GTA V, Elden Ring, Hogwarts Legacy) для выбранной связки GPU + CPU.

## Сценарий 2. Работа от администратора

### 2.1. Вход в админ-панель
- Нажмите «⚡ Админ» в навигационной панели или перейдите по адресу: http://127.0.0.1:5000/admin

### 2.2. Просмотр всех комплектующих (/admin)
- Таблица со всеми товарами: ID, название, категория, сокет, TDP, цена.
- Для каждого товара доступны кнопки:
  - ✏️ — редактировать
  - 🗑️ — удалить

### 2.3. Добавление комплектующего (/admin/add)
1. Нажмите «+ Добавить комплектующее».
2. Заполните форму:
   - **Категория** — выберите из списка (CPU, GPU, RAM и т.д.)
   - **Название** — полное название товара
   - **Цена** — в рублях
   - **TDP** — энергопотребление в ваттах
   - **Производительность** — условный балл (0–100), используется для прогноза FPS
   - **Сокет** — указать для CPU и материнских плат (AM5, LGA1700), для остальных — оставить пустым
3. Нажмите «Добавить».
4. Товар появится в каталоге и конструкторе.

### 2.4. Редактирование комплектующего (/admin/edit/<id>)
1. Нажмите ✏️ напротив нужного товара.
2. Измените нужные поля.
3. Нажмите «Сохранить».

### 2.5. Удаление комплектующего (/admin/delete/<id>)
1. Нажмите 🗑️ напротив товара.
2. Подтвердите удаление в диалоговом окне.
3. Товар удалён из базы данных.

### 2.6. Массовое обновление цен
1. Перейдите в каталог (/catalog).
2. Нажмите «⟳ Обновить все цены».
3. Приложение обращается к интернет-магазину regard.ru и обновляет цены всех товаров.
4. Результат отображается в уведомлении: «Обновлено N из M цен».
